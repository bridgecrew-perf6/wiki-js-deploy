# 一键安装Wiki.js

## 一、前提

- 1.确保 Docker 已经安装，并且已经处于运行状态
- 2.确保网络顺畅，能够访问 Github


## 二、使用

### 1.下载数据

- 方法1：`git clone ssh://git@192.168.123.204:4422/Docker/wiki-js-deploy.git`
- 方法2：[进入这里手动下载解压](http://gitlab.heshixi.loc/Docker/wiki-js-deploy)
